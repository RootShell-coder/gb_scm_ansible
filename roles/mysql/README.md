# MySQL Role

## Role Variables

| Name          | Description                  | Default         | Required |
| ------------- | ---------------------------- | --------------- | :------: |
| root_password | Password for root user in DB | `root`          |    no    |
| db_name       | database name                | `wordpress`     |    no    |
| db_user       | Username                     | `wordpressuser` |    no    |
| db_pass       | Password                     | `wordpresspass` |    no    |
